# 言寺资产管理系统 - 后端

Spring Boot 后端服务，提供资产管理系统的 REST API。

## 技术栈

- Spring Boot 3.5.11
- MyBatis 3.0.3
- MySQL
- JWT 认证
- Apache POI (Excel导出)

## 快速开始

1. 创建数据库 `yansiams`
2. 执行 `src/main/resources/schema.sql` 初始化表结构
3. 修改 `application.yml` 数据库配置
4. 运行 `YansiAmsSpringApplication`
